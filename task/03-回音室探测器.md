# 🕵️ 回音室探测器（Echo Chamber Radar）

## 场景描述

一键扫描你的时间线，生成「信息饮食报告」：

```
📊 你的 X 信息饮食报告

🥗 话题营养均衡度：❌ 极度偏食
  - 科技相关：72% ← 过量
  - 金融投资：18% ← 正常
  - 政治社会：5%  ← 严重不足
  - 文化艺术：3%  ← 几乎为零
  - 生活方式：2%  ← 缺失

🪞 你可能以为自己观点多元，实际上：
  你 87% 的信息来自同一种声音
  你关注的人中，91% 持相似立场

🚨 你正在看不见的世界里原地打转

💡 建议关注：
  @some_culture_person   ← 补充文化视角
  @some_politics_person  ← 打破政治盲区
  
「一个只吃汉堡的人，不会知道自己缺维生素」
```

## 为什么有趣

- **认知冲击**——每个人都以为自己「观点开放」，但数据告诉他不是，这种反差极具冲击力
- **社交货币**——「我的信息饮食报告」天然适合截图分享，比任何功能都更有传播力
- **哲学深度**——触及了「信息茧房」这个时代级话题，自带思想价值

## 技术实现

- 扫描最近 200 条时间线推文
- 按话题分类（AI 通过 ChatClient）
- 计算分布，生成报告
- 不需要复杂算法，ChatClient 就够

## 交互设计要点

- 侧边栏或浮动面板触发扫描
- 报告用可视化图表（饼图/雷达图）呈现
- 一键生成分享图片

## 优先级

P2 — 传播力最强，但需要较多时间线数据采集
