# 🎪 热评生成器（Hot Take Generator）

## 场景描述

看到一条推文，想写犀利回复但想不到好角度——AI 给你 3 个「热评方案」：

```
对 @someone 的推文：「AI 已经超越了人类创造力」

🔥 3 条热评方案：

方案 A（反讽型）😏
「是啊，AI 超越了人类的创造力，
  但还没超越人类的幻觉能力」

方案 B（数据反击型）📊  
「超越了吗？截至 2024 年，
  AI 生成的画作在专业评审中 0 次获奖，
  人类：4,217 次」

方案 C（哲学追问型）🤔
「有意思的问题：什么是"超越"？
  如果 AI 只是更快地组合已有元素，
  它超越的是效率，不是创造力」

💡 选择建议：
  想引发讨论 → 选 B
  想获得 likes → 选 A
  想显示深度 → 选 C
```

## 为什么有趣

- **参与欲**——X 的核心是互动，但大多数人只看不说，因为不知道怎么说得有趣
- **社交表现欲**——好的回复能带来 followers，这是真实的社交回报
- **游戏感**——选择不同风格的回复像选角色，每次体验不同

## 技术实现

- 在推文回复框旁添加「AI 热评」按钮
- 提取原推文内容，通过 ChatClient 生成多风格回复
- prompt 设计：反讽型、数据型、哲学型等不同风格模板
- 一键插入到回复编辑框

## 交互设计要点

- 回复输入框旁的按钮触发
- 3 个方案用卡片式呈现，可点击选中
- 选中后直接插入到回复框，用户可再微调
- 生成速度要快（< 3 秒），否则打断回复冲动

## 优先级

P0 — 最低门槛、最高频次、最容易落地
