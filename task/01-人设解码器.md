# 🎭 人设解码器（Persona Decoder）

## 场景描述

鼠标悬停在任意用户头像上，AI 立刻生成一段「人设分析」：

```
@elonmusk 的人设画像：
━━━━━━━━━━━━━━━━━━━
🎭 角色类型：技术狂人 + 段子手混合体
🔥 核心叙事："我在拯救世界，顺便发 meme"
🪞 自我认知 vs 外界认知：自认为颠覆者 / 被视为搅局者
📈 情绪曲线：85% 亢奋 → 10% 暴怒 → 5% 突然感性
🎯 真实意图概率：40% 真的想讨论 → 60% 在钓鱼/测试舆论
💡 一句话总结：每个看似随意的 tweet 都是精心设计的舆论实验
```

## 为什么有趣

- **窥探欲**是人性最强驱动力——每个人都想「看穿别人」
- X 上人人都在「表演」，但没人承认——你帮他们揭穿，这就是快感
- 分析结果自带传播性：「看看 AI 怎么分析你的人设！」——用户会把结果截图分享

## 技术实现

- 抓取用户最近 50 条推文
- 通过 ChatClient 发给 AI，使用结构化 prompt 输出画像
- 成本约 0.01 元/次，边际成本极低

## 交互设计要点

- 悬停触发，卡片式浮层展示
- 一键截图分享（自带品牌水印）
- 支持刷新重新分析（推文更新后）

## 优先级

P1 — 传播性强，成本低，易于落地
